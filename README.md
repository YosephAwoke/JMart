# JMart

Premium MERN e-commerce foundation for the Ethiopian market.

## Apps

- `apps/web`: Vite + React storefront
- `apps/api`: Express + MongoDB API
- `packages/shared`: locale data, types, and domain constants

## Scripts

- `npm run dev:web`
- `npm run dev:api`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
